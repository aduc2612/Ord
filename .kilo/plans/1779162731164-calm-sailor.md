## Phase 1: Supabase Database Setup (Manual Steps)

These steps must be done in the Supabase SQL Editor by the operator.

### Step 1.1: Create `powersync_role` and Publication
Run this SQL in Supabase SQL Editor:

```sql
-- Create replication role for PowerSync
CREATE ROLE powersync_role WITH REPLICATION BYPASSRLS LOGIN PASSWORD '<secure-random-password>';
GRANT SELECT ON ALL TABLES IN SCHEMA public TO powersync_role;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT SELECT ON TABLES TO powersync_role;

-- Create publication (must be named "powersync")
CREATE PUBLICATION powersync FOR ALL TABLES;
```

**Note**: The operator needs to replace `<secure-random-password>` with a strong password and save it.

### Step 1.2: Set REPLICA IDENTITY on tables
PowerSync needs `REPLICA IDENTITY FULL` for DELETE operations:

```sql
ALTER TABLE public.test_items REPLICA IDENTITY FULL;
```

## Phase 2: PowerSync Cloud Setup (Mixed Manual + CLI)

### Step 2.1: Prerequisites (Operator Action)
The operator needs:
1. A PowerSync Cloud account at https://dashboard.powersync.com/
2. A PowerSync project created (if not already exists)
3. The Project ID from the dashboard
4. A PowerSync Personal Access Token (PAT) from https://dashboard.powersync.com/account/access-tokens

### Step 2.2: Install PowerSync CLI
```bash
npm install -g powersync@latest
```

### Step 2.3: Add credentials to `.env`
Add these to `.env`:
```
PS_DATABASE_URI=postgresql://powersync_role:<password>@db.urzbmgqlmmdmwijvcfyt.supabase.co:5432/postgres
POWERSYNC_URL=https://<instance-id>.powersync.journeyapps.com
```

The `PS_DATABASE_URI` uses the Supabase direct connection string with the `powersync_role` credentials. The operator gets the connection string from Supabase Dashboard → Connect → Direct connection, replacing `[YOUR-PASSWORD]` with the `powersync_role` password.

### Step 2.4: Initialize PowerSync Cloud Config
```bash
powersync login
powersync init cloud
```

This creates `powersync/` directory with `service.yaml`, `sync-config.yaml`, and `cli.yaml`.

### Step 2.5: Configure `service.yaml`
Edit `powersync/service.yaml`:

```yaml
replication:
  connections:
    - type: postgresql
      hostname: db.urzbmgqlmmdmwijvcfyt.supabase.co
      port: 5432
      database: postgres
      username: powersync_role
      password:
        secret: !env PS_DATABASE_PASSWORD
      sslmode: verify-full

client_auth:
  supabase: true
```

Add `PS_DATABASE_PASSWORD=<the-powersync_role-password>` to `.env`.

### Step 2.6: Configure `sync-config.yaml`
Edit `powersync/sync-config.yaml`:

```yaml
config:
  edition: 3

streams:
  user_data:
    auto_subscribe: true
    queries:
      - SELECT * FROM test_items WHERE user_id = auth.user_id()
```

### Step 2.7: Link and Deploy
```bash
powersync link cloud --create --project-id=<project-id>
powersync validate
powersync deploy
```

The instance ID is printed after `link cloud --create`. Construct `POWERSYNC_URL=https://<instance-id>.powersync.journeyapps.com` and add to `.env`.

### Step 2.8: Enable Supabase Auth in Dashboard (Manual)
1. Go to PowerSync Dashboard → Client Auth for the instance
2. Enable "Use Supabase Auth" checkbox
3. Leave JWT Secret field empty (PowerSync auto-detects with new signing keys)
4. Click "Save and Deploy"

## Phase 3: App Integration

### Step 3.1: Install PowerSync Packages


### Step 3.2: Create PowerSync Connector (`src/lib/powersync.ts`)
Create the backend connector with `fetchCredentials()` and `uploadData()`:

```ts
import { supabase } from './supabase';
import type { PowerSyncBackendConnector, PowerSyncCredentials } from '@powersync/react-native';
import { AbstractPowerSyncDatabase, UpdateType } from '@powersync/react-native';

const POWERSYNC_URL = process.env.EXPO_PUBLIC_POWERSYNC_URL ?? '';

export const connector: PowerSyncBackendConnector = {
  async fetchCredentials(): Promise<PowerSyncCredentials> {
    const { data: { session }, error } = await supabase.auth.getSession();
    if (error || !session) throw error ?? new Error('No session');
    return {
      endpoint: POWERSYNC_URL,
      token: session.access_token,
      expiresAt: new Date(session.expires_at! * 1000),
    };
  },

  async uploadData(database: AbstractPowerSyncDatabase): Promise<void> {
    const transaction = await database.getNextCrudTransaction();
    if (!transaction) return;

    try {
      for (const op of transaction.crud) {
        const { op: opType, table, opData, id } = op;
        let result: { error: unknown };
        if (opType === UpdateType.PUT) {
          result = await supabase.from(table).upsert({ ...opData, id });
        } else if (opType === UpdateType.PATCH) {
          result = await supabase.from(table).update(opData).eq('id', id);
        } else {
          result = await supabase.from(table).delete().eq('id', id);
        }
        if (result.error) throw result.error;
      }
      await transaction.complete();
    } catch (error: unknown) {
      const err = error as { code?: string; status?: number };
      const isPermanent = err?.code === '42501' || err?.status === 400;
      if (isPermanent) {
        console.error('Permanent upload error, skipping:', error);
        await transaction.complete();
        return;
      }
      throw error;
    }
  },
};
```

### Step 3.4: Create PowerSync Provider (`src/providers/PowerSyncProvider.tsx`)
Wrap the app with PowerSync context, connecting after auth:

```tsx
import { ReactNode, useEffect, useState } from 'react';
import { PowerSyncContext } from '@powersync/react-native';
import { powerSyncDb } from '@/lib/powersync-db';
import { connector } from '@/lib/powersync';
import { supabase } from '@/lib/supabase';

export function PowerSyncProvider({ children }: { children: ReactNode }) {
  const [isReady, setIsReady] = useState(false);

  useEffect(() => {
    const { data: { subscription } } = supabase.auth.onAuthStateChange((_event, session) => {
      if (session) {
        powerSyncDb.connect(connector);
      } else {
        powerSyncDb.disconnectAndClear();
      }
    });

    supabase.auth.getSession().then(({ data: { session } }) => {
      if (session) {
        powerSyncDb.connect(connector);
      }
      setIsReady(true);
    });

    return () => subscription.unsubscribe();
  }, []);

  if (!isReady) return null;

  return (
    <PowerSyncContext.Provider value={powerSyncDb}>
      {children}
    </PowerSyncContext.Provider>
  );
}
```
